# كرتون زمان — مشروع عرض مسلسلات كرتونية قديمة

مشروع ثابت بسيط لعرض مسلسلات كرتونية قديمة باللغة العربية. يحتوي المشروع على HTML/CSS/JS بسيط وملف بيانات JSON.

تشغيل محلي:
1. استعمل خادم محلي: `python -m http.server 8000`
2. افتح المتصفح: http://localhost:8000

ملاحظات قانونية:
